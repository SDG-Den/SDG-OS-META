Package Name: sdgos-meta
Descriptive Name: SDG OS Metapackage
Source: https://git.sdgcloud.nl/SDGDen/SDG-OS-META
Maintainer: SDGDen <sdgden@sdgcloud.nl>
Version:0.2

Dependencies: 
sdgpkg (bootstrap required), git, curl

Description: 
Minimal Bash metapackage that installs 12 child SDG-OS packages in sequence via sdgpkg install. Convenience metapackage for setting up a complete SDG-OS desktop in one command. Installs: unipkg, sdg-fetch, sdg-docs, sdg-mango, sdg-tips, sdg-util, sdg-themes, sdg-wayshell, sdg-wayshell-conf, sdg-vox, sdg-glyphs, sdg-mango-helpers.
